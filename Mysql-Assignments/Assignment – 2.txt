Q1) Create the tables Member, Books, and Issue WITHOUT constraints.

Ans:   1) MEMBER Table :
       Query: create table Member(
    -> Member_Id INT COMMENT 'Unique Member ID',
    -> Member_Name VARCHAR(30) COMMENT 'Name of the Library member',
    -> Member_Address VARCHAR(50) COMMENT 'Address of the member',
    -> Acc_Open_Date DATE COMMENT 'Date of membership',

    -> Penalty_Amount DECIMAL(10,2) COMMENT 'Penalty amount due');


   2) BOOKS Table :
    Query:  create table Books(
    -> Book_No INT COMMENT 'Book identification number',

    -> Cost DECIMAL(10,2) COMMENT 'Cost of the book',
    -> Category VARCHAR(15) COMMENT 'Science/Fiction/Database/RDBMS/Others');


   3) ISSUE Table : 
   Query: create table Issue(
    -> Lib_Issue_ID INT COMMENT 'Library Issue ID',
    -> Book_No INT COMMENT 'Book number issued',
    -> Member_ID INT COMMENT 'Member who issued the book',
    -> Issue_Date DATE COMMENT 'Date of issue',
    -> Return_Date DATE COMMENT 'Date of return');
